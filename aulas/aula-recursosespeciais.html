A tag datalist usa a tag option apenas como sugestão para digitação, diferente da tag select.

A tag kbd serve para mostrar inputs de teclado. Ex.: CRTL + C

A tag code serve para mostrar comandos com fonte mono-space. Ex.: background-color

A tag pre serve para colocar todo o código CSS no texto com fonte mono-space. Ex.:

            code {
                background-color: #f1f1f1;
                color: red;
            }
Pra que serve a tag details?

A tag details serve para criar um campo que se expande ao ser clicado.

Pra que serve a tag summary?

A tag summary serve para criar um título para a tag details.

A tag meter é um medidor.

A tag progress serve para definir o progresso de uma ação.

A tag canvas é usada para manipular gráficos dentro dela usando o javascript.
