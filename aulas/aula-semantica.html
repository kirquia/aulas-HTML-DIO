HTML - Semântica

Tags que, ao serem inseridas, ficam em vermelho estão em desuso no HTML 5. Ex.: bgcolor, font, align.

Acessibilidade

A tag alt serve para robôs de acessibilidade lerem uma descrição previamente escrita nela afim de facilitar o acesso a ela por pessoas com dificuldade.

Web Scraping

"Rastreador" (Crawler - também chamado de "robô" ou "indexador") é um termo genérico para qualquer programa usado para descobrir e examinar sites automaticamente seguindo links entre páginas da Web. Ex.: GoogleBot
